为视频播放器启用更多的快捷键:
- <kbd>w</kbd> 网页全屏
- <kbd>t</kbd> 宽屏
- <kbd>m</kbd> 静音
- <kbd>d</kbd> 弹幕开关
- <kbd>l</kbd> 点赞
- <kbd>c</kbd> 投币
- <kbd>s</kbd> 收藏
- <kbd>j</kbd> 长前进
- <kbd>Shift + j</kbd> 长倒退
- <kbd>Shift + w</kbd> 稍后再看
- <kbd>Shift + ↑/↓</kbd> / <kbd>Shift + ,/.</kbd> 播放速度调整
- <kbd>Shift + /</kbd> 重置播放速度
